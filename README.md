## What This Skill Does

| Workflow | Calls | Revenue Impact |
|----------|-------|---------------|
| Campaign Lifecycle | 4-5 | **Direct: drives pipeline** |
| Audience Building | 2-3 | Targeting precision |
| Content Creation | 1-2 | Engagement quality |
| Channel Execution | 1-2 | Multi-channel reach |
| Performance | 2-3 | Optimize spend |
| Budget Governance | 1-2 | Prevent overspend |

### Without this skill:
- Campaigns launched without budget check
- No cross-channel attribution
- Audiences built manually
- Performance reviewed weekly (too late)

### With this skill:
- Budget verified before every launch
- Multi-touch attribution across channels
- Audiences built from criteria in 1 call
- Real-time ROAS tracking

## Installation

```bash
git clone https://github.com/zavora-ai/skill-marketing-orchestration.git ~/.skills/skills/marketing-orchestration
```

## Requirements

**Required:** `mcp-marketing` (22 tools)
**Cross-MCP:** `mcp-crm` (lead data), `mcp-analytics` (funnel metrics), `mcp-slack` (performance alerts)

## Success Criteria

| Metric | Target |
|--------|--------|
| Campaign speed | Draft to launch in 5 calls |
| Budget governance | Never exceed without approval |
| Attribution | Multi-touch across all channels |
